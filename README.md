# Focus-is-All-You-Need
Medallion architecture pipeline analysing Ford's 14% BEV sales decline. Ingests DMV, AFDC, CarGurus and Google Trends data across Bronze/Silver/Gold layers. PCA clustering, Random Forest feature importance, and NVIDIA LLM RAG (llama3-70b, mixtral-8x7b) to identify distribution, portfolio and GTM gaps.
